# SoccerActionEnv
A data-driven soccer environment built using event data. Compatible with OpenGYM API.

## Instalation
    - Requires Python 3.7 due to Tensorflow 1 requirements on stable_baselines3
    - Run pip install -r requirements.txt
    - Done!
